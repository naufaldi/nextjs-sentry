# NextJS and Sentry

Learning example NextJs and Sentry for Error Monitoring

## Installation

Go to project

```bash
  npm install
```

Run project

```bash
    npm run dev
```

## Lessons Learned

How integrated NextJS and Sentry for Monitoring error in production and development.
